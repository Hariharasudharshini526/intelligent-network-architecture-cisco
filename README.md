# Intelligent Network Architecture using Cisco Packet Tracer

## Overview

This project demonstrates the design and implementation of an enterprise-level intelligent network architecture using Cisco Packet Tracer.

The network is designed to provide secure, scalable, and efficient communication between multiple departments using VLAN segmentation, Inter-VLAN routing, Access Control Lists (ACLs), and centralized monitoring services.

The architecture also integrates IoT devices for smart monitoring and includes network management services such as Syslog and Network Time Protocol (NTP).

## Network Topology

The network consists of:

- Core Router
- Layer-3 Switch
- Layer-2 Switches
- PCs
- Servers
- IoT Devices
- DHCP Server
- Syslog Server
- NTP Server

Each department is placed in an independent VLAN to improve security and performance.

3. Start Simulation Mode.
4. Test communication between VLANs.
5. Verify ACL rules.
6. Observe traffic flow.
